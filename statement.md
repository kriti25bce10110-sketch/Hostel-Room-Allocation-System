# Problem Statement
Hostel room allocation in many institutions is still handled manually, resulting in duplication, errors, and lack of clarity for students and administrators. A simple digital system is required to manage student registrations and room assignment efficiently.

# Scope of the Project
- Register students with basic details
- Allocate rooms based on gender and availability
- Display reports for allocated/available rooms
- Basic Python implementation using loops and conditionals only

# Target Users
- Hostel administration
- Students seeking room information

# High-Level Features
- Student data input and storage
- Gender-based room allocation
- Availability tracking
- Reporting module
