# Hostel Room Allocation System

# Project Title
Hostel Room Allocation System

# Overview of the Project
This project is a simple, menu-driven Python application designed to automate hostel room allocation for students. It supports:
- Student registration
- Gender-based room allocation
- Availability tracking
- Reporting of allocations

It is built using only the basic Python constructs(if/elif, loops, lists, dictionaries) as per academic project requirements.


# Features
- Student Registration – Add students with name, roll number, and gender
- Gender-based Room Allocation – Automatically assigns rooms for boys/girls
- Duplicate Check – Prevents double registration or allocation
- Room Availability Report – Shows free and allocated rooms
- Student Allocation Report – Displays which student got which room
- Simple Console UI – Easy to use and understand


# Technologies / Tools Used
- Python 3.x
- Console (Terminal) interface
- Basic Python data structures:
  - Lists
  - Dictionaries
  - Loops
  - Conditionals (if/elif/else)

# Instructions for Testing
1. Run the program.
2. Choose 1 to register a student.
   - Try entering a few test names.
3. Choose 2 to allocate a room.
4. View the results using 3. View Reports.
5. Try entering duplicate roll numbers to confirm the system blocks duplicates.
